Deploy full stack project into cloud

Posted on 2019-01-14 | In web

Forget to renew… Have to deploy again…

A. 管理控制台

管理控制台：客户端xshell使用密钥或者账号密码进行连接，还是账号密码方便（得删除密钥才行！！）

B. 下载Node.js和nginx, mongodb

  • Node.js

  • nginx
    所有的配置文件都在 /etc/nginx下，每个虚拟主机安排在/etc/niginx/sites-available下
    实际的web内容在目录/var/www/html下

  • MongoDB
    mongodb
    sudo service mongod start
    sudo service mongod restart
    默认mongodb地址
    /var/lib/mongodb

C. 修改主机名

修改linux主机名
vim /etc/hostname
还得修改
vim /etc/host来避免 sudo: unable to resolve host leiger的问题解决方案

端口管理
阿里云默认开启的端口不多，自己的node.js3000端口没打开，外网无法访问
查看端口调用lsof -i:4000
杀端口kill -9 (pid)

ctrl+c: 端口被关闭
ctrl+z: 后台会运行

pm2更好用
npm install pm2 -g
pm2 start xx
pm2 stop xx
pm2 start all
pm2 stop all
pm2 log
pm2 list
mv www demo-server 改名字

D. 二级域名部署

阿里云服务器上得先添加域名规则先
重启nginx
nginx -s reload

在导入json文件到mongodb中时，总是出现错误:
unreachable server
不知道为啥试了各种方法都不行，重启下mongodb就好了…
sudo service mongod restart
还得注意下数据库名称，名称应该为vue_mall而不是vmall

https的话阿里云之前申请过了，这次只要下载证书和密钥就可以了
得在nginx上配置才行
配置主要代码

强制https访问
